## 1. 회사 개요

**EO Studio**는 한국·미국·베트남 3개 법인을 운영하는 글로벌 미디어 스타트업입니다.

| 항목 | 내용 |
|------|------|
| **한국법인** | 주식회사 이오스튜디오 — 강남구 압구정로 28길 9-2, 4층 |
| **미국법인** | EO STUDIO INC — SF EO House |
| **베트남법인** | EO Vietnam (FDI 외국인투자법인) — 호치민 기반 |
| **CEO** | 김태용 |
| **CCO** | 유건욱 (Gunwook Yoo) |
| **매출 목표** | 2026년 글로벌 $7M |
| **Slack** | eo2019.slack.com |

### 주요 사업 영역
- **YouTube 콘텐츠**: EO Korea + EO Global 채널 (Premium 심층 인터뷰 + Standard 주간 포맷 2-tier)
- **BOLD Seoul**: 글로벌 컨퍼런스
- **B2B 멘토링**: 기업 대상 멘토링/교육
- **글로벌 미디어**: 영어권 콘텐츠 확장
- **EO Magazine** (eomag.io): 글로벌 텍스트 매거진
- **EO School (이오스쿨)**: 온라인 교육 플랫폼
- **이오플래닛**: 한국 자체 미디어 플랫폼 (83,000+ 유저, 15,000+ 아티클)
- **EO Asia**: 베트남 기반 아시아 채널 (신규)

## 7. 지출 & 경비 처리

### 지출 요청 프로세스
1. **Airtable 양식** 작성 → [지출요청 양식](https://airtable.com/apphaMgCZMSN3ysHk/pagg4CzxDLLtYRbjq)
2. 양식 제출 시 `#request-지출요청` 채널에 자동 알림
3. 운영팀 확인 후 **매주 금요일** 일괄 이체
4. 이체 완료 시 "입금 완료" 알림

### 양식 필수 항목
| 항목 | 설명 |
|------|------|
| **항목** | 지출 내역 설명 |
| **분류** | 개인 지출 / 업체 구매 / 외주 인력 / 운동비 |
| **연관 프로젝트** | 해당 프로젝트명 |
| **금액** | 원화(KRW) 또는 달러(USD) |
| **구매 링크** | 있으면 첨부 |
| **특이사항** | 법인카드 한도 제한 등 참고사항 |

### 주의사항
- **제출 마감**: 목요일까지 제출 → 금요일 이체 가능
- **법인카드 한도 초과**: 개인카드로 결제 후 지출요청에 "법인카드 한도 제한으로 개인카드 구매" 기재
- **교육비**: 개인카드로 선결제 → 교육 완료 후 Airtable에 후기 작성 필수 → 지출요청
- **미국 법인 송금**: **Zelle** 사용 (Venmo 불가), Brex 법인카드 사용
- **인보이스 발행**: Bill.com (US), 청구스 (KR) — `#all-프로젝트현황`에서 청구 상태 자동 확인

### 운동비 (운동해EO)
- 월 10만원 운동비 지원 (챌린저스 앱 연동)
- 챌린저스 앱 가입 시 1,000원 결제 필요 (한국 계좌)
- 미국 거주자는 운영팀에 별도 문의

## 8. 외주 계약 요청

### 프로세스
1. `#request-외주계약요청` 채널에서 프리랜서 계약요청 폼 제출
2. 접수 후 ASH(안서현)가 계약서 처리
3. **새로운 외주업체**: ASH 승인 필요
4. **반복 프리랜서**: 최초 계약서 1장만 신청, 금액/조건 변경 시 재계약
5. **프리랜서 정산**: Airtable 연동

### 폼 필수 항목
```
요청자 / 담당자
이름, 이메일, 휴대폰
프로젝트명
담당업무
계약금액
계약기간
추가요청사항
```

| 유형 | 단가 기준 |
|------|-----------|
| 썸네일 디자이너 | 건당 40,000원 |
| 컷편집 | 월 정액 또는 건당 |

## 9. 콘텐츠 제작 워크플로우

### 영상 제작 플로우
```
인터뷰 → 스토리라인 작성 → 편집 → Frame.io 피드백 → 수정 → 발행
```

### 핵심 원칙
- **인터뷰 직후 24시간 내** 편집 시작 (기억 감퇴 방지)
- **B-roll**은 사전 요청
- **IP 구분 중요**: EO Korea vs EO Global — 작업자 배정과 채널이 다름
- **숏폼**: 촬영 후 3개씩 묶어 스케줄 업로드
- **지적 긴장감(intellectual tension)**: 콘텐츠의 핵심 가치

### 콘텐츠 전략 (2-tier)
| 티어 | 설명 |
|------|------|
| **Premium** | 심층 인터뷰 (a16z GP, Kalshi, TestBox 등 글로벌 파운더) |
| **Standard** | 주간 포맷 콘텐츠 (Thinking Mode 등 시리즈) |

### 촬영 일정
- `#all-타임테이블` 채널에서 촬영 일정 공지 확인
- 촬영 장소: 성수 스튜디오, 역삼동 스튜디오 등
- 장비: Gear List (Google Sheets) 참고
- 촬영 당일 세팅 시간에 맞춰 도착 (메이크업 일정 포함)

### 편집 도구
- **Final Cut Pro** + **Red Giant** 플러그인
- **Frame.io**: 영상 피드백 플랫폼 (상세 피드백을 남기는 것이 중요)
- AI 활용: 인터뷰 질문 스코어링/시뮬레이션, 카드뉴스 Gamma API 자동 생성

### 한국 미디어 (번역/로컬라이징)
- 글로벌 콘텐츠 → 한국 번역은 **랭킹모드 우선** 발행 (시의성 중요)
- 팩트체크 철저히 — 인물 직함/역할 변경 여부 반드시 확인
- 너무 늦게 올리면 "철지난 정보를 최신것처럼 이야기하는 채널"로 보일 위험
