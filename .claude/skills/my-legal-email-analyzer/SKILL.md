# my-legal-email-analyzer

변호사 메일 스레드를 분석하여 요점 추출 + 답장 초안 자동 생성하는 스킬.
Lisa 법률 어드바이저와 연동하여 초안 검토까지 한 번에 처리.

## 사용 시나리오
- SEUM 변호사 이메일 스레드 분석 → 질문 항목 추출 → 답장 초안 생성
- Eugene Kang 변호사 메일에서 요청사항 추출 → 액션 아이템 정리
- 법률 문서 관련 메일 히스토리 조회 → 의사결정 타임라인 정리

## 실행 흐름

### Step 1: 메일 스레드 수집
```
사용자 입력: 검색 쿼리 또는 스레드 ID
→ Gmail API로 스레드 전체 조회
→ 시간순 정렬 + 발신자별 분류
```

### Step 2: 요점 추출 (병렬)
```
Agent 1: question-extractor — 상대방이 한 질문/요청 추출
Agent 2: decision-tracker — 확정된 결정사항과 미확정 이슈 분리
```

### Step 3: 답장 초안 생성
```
추출된 질문/요청에 대해 답장 초안 자동 생성
→ Lisa 어드바이저 Mode A로 법률 검토
→ 수정 반영된 최종 초안 출력
```

### Step 4: 사용자 확인
```
초안 출력 → 사용자가 수정/확인 → 수동 발송 (자동 발송 금지)
```

## Lisa 어드바이저 연동
- 법률 판단이 필요한 항목은 자동으로 Lisa Mode A로 라우팅
- "확인 필요" 항목은 에스컬레이션 표시

## 상태: 초안 (구현 대기)
- 현재 수동으로 Gmail 검색 → 읽기 → Lisa 호출 → 초안 작성 반복
- Day 5-6에서 SEUM 메일 처리 시 이 패턴이 3회 이상 반복됨
