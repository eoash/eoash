# EO Studio 매출 대시보드 설계

**작성일**: 2026-02-23
**데이터 소스**: Google Sheets (월별 현황 보고 양식_(주)이오스튜디오)
**플랫폼**: Streamlit 멀티페이지

---

## 1. 프로젝트 구조

```
dashboard/
├── app.py                  # 홈 (Overview) 페이지
├── pages/
│   ├── 1_매출.py           # 매출 분석
│   ├── 2_현금.py           # Cash Position
│   └── 3_AR.py             # 미수금 관리
└── utils/
    ├── sheets.py           # 구글시트 데이터 로딩 + 캐싱
    └── charts.py           # 차트 헬퍼
```

---

## 2. 페이지별 내용

### 홈 (app.py)
- KPI 카드 4개: 2026 YTD 총매출 / 목표 달성률 / Cash Position 합계 / 미회수 AR 합계
- 최근 3개월 매출 미니 차트

### 1_매출.py
- 월별 실적 vs 목표 막대 그래프 (2026)
- BU별 매출 비중 파이차트 (KR한국 / 플래닛한국 / GL글로벌)
- 연도별 추이 라인차트 (2024 / 2025 / 2026)

### 2_현금.py
- 국가별(한국/미국/베트남) 잔액 카드
- 월별 유입/유출 차트
- Total 누적 잔액 추이

### 3_AR.py
- 전체 미수금 합계 카드
- 거래처별 미수금 테이블 (금액 내림차순)
- 회수일수 분포 히스토그램

---

## 3. 기술 스택

- **streamlit** — 대시보드 프레임워크
- **plotly** — 차트 (막대/파이/라인)
- **pandas** — 데이터 가공
- **google-api-python-client** — 구글시트 API (기설치)

---

## 4. 데이터 흐름

- `utils/sheets.py`에서 Google Sheets API 호출
- `@st.cache_data(ttl=300)` — 5분 캐싱
- 사이드바 "새로고침" 버튼 → 캐시 초기화 후 재로딩

### 데이터 파싱 규칙
- 금액: `"1,611,557"` → 콤마 제거 후 int 변환
- 원화: `"₩141,753,433 "` → `₩`와 공백 제거 후 변환
- 오류값: `"#DIV/0!"` → 0 처리

---

## 5. 구글시트 매핑

| 페이지 | 시트명 | 주요 컬럼 |
|--------|--------|-----------|
| 홈/매출 | `2026 매출` | BU, 카테고리, 1월~12월, 합계, 목표, 달성율 |
| 홈/매출 | `매출비교` | 연도, 1월~12월, 합계, 인원수, 인당매출 |
| 현금 | `2026 Cash Position Summary` | 국가, Balance, Inflows, Outflows, Net Change |
| AR | `거래처별 미수금/회수기간 관리표` | 상호, 공급가액, 작성일자, 정산일자, 회수일수 |
