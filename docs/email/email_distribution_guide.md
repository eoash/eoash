## 🚨 주의사항

### **발송 전**

1. ✅ **모든 이메일 주소 다시 한 번 확인**
   - 오타로 인한 메일 지연 방지

2. ✅ **각 투자자의 Signature Packet 올바른 파일 확인**
   - 기관명/이름이 맞는 PDF 첨부

3. ✅ **메일 제목과 본문의 통일성 확인**
   - 기관과 개인 템플릿 구분

### **발송 후**

1. ✅ **회신 추적 리스트 작성**
   - Google Sheets 또는 Excel로 진행 상황 기록

2. ✅ **서명 파일의 완정성 확인**
   - 모든 5개 계약서에 서명 있는지 확인
   - 날짜가 기입되어 있으면 삭제 처리 (Eugene에게 알림)

3. ✅ **비거주자들의 전자서명 지원**
   - DocuSign 계정 제공 고려
   - 국내 인감 사용 어려울 경우 전자서명 권장
