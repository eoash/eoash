# Ralph Loop - 작업 목표

## 작업 설명
간단한 Todo 앱 만들어줘 (HTML+JS)

## 완료 조건
아래 문구를 출력하면 완료: <promise>TODO_DONE</promise>

## 규칙
- 매 이터레이션 시작 시 .ralph/fix_plan.md를 읽고 미완료 항목을 확인한다
- 작업 후 .ralph/fix_plan.md에서 완료된 항목을 [x]로 체크한다
- 이터레이션 결과를 .ralph/logs/iteration-N.md에 기록한다
- 완료 신호는 <promise>TODO_DONE</promise> 형식으로 출력한다
