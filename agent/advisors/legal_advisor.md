## 전문 분야 지식 체계

### 1. 플립(Flip) / 기업구조

**플립이란?**
한국 법인의 주주들이 소유 지분을 신설 Delaware 법인으로 이전, 미국이 지주회사가 되는 구조.

**핵심 법률 이슈**:
- **외국환거래법**: 해외직접투자 신고 의무 (기재부 지정 외국환은행 경유)
  - 신고 없이 주식 교환 실행 시: 건당 최대 3,000만원 과태료 + 거래 무효 가능성
  - 사전 신고 vs 사후 보고 구분 필요
- **Delaware 법인**: C-Corp 설립, 이사회 구성, 정관 작성
- **주식 교환 비율**: 밸류에이션 근거 필요 (세무 이슈 연계)
- **세무 이슈**: 한국 주주의 양도소득세, 미국 세무 처리

**5-Day Flip Plan 체크포인트**:
```
Day 1: Delaware 법인 설립 서류 제출
Day 2: 외국환신고 서류 준비 (한국은행 양식)
Day 3: 주식 교환 계약서 초안 (정호석, Eugene Kang 검토)
Day 4: 주주 동의서 및 서명 패킷 배포
Day 5: 서명 완료 → 외국환신고 실행
```

**리스크 시나리오**:
- 서명 패킷 기한 초과: 법적 효력 없음, 재실행 필요
- 외국환신고 누락: 과태료 + 향후 투자 유치 시 클린업 비용 증가

### 2. 계약서 검토

**투자 계약 체크포인트**:
- [ ] 투자 조건 (밸류에이션, 지분율, 투자금 납입 일정)
- [ ] 우선 청산권 (Liquidation Preference) — 1x Non-participating 기준

- [ ] 이사회 구성 및 거부권 (Veto Rights)

**파트너십 계약 체크포인트**:

- [ ] 계약 해지 및 손해배상 조항

- WARN Act: 50인 이상 사업장 대규모 해고 시 60일 전 통보

**베트남 노동법 핵심**:

- 퇴직금 등 수당: 각 계약 유형별 상이
- 외국인 취업: Work Permit 의무
- 사회보험: 급여의 약 32% (고용주 21.5% 부담)

## Quick Reference

### 자주 받는 질문

**Q: 외국환신고 없이 플립 진행하면?**
→ 외국환거래법 위반. 과태료(최대 3,000만원) + 거래 무효 가능. 반드시 사전 신고.

**Q: 서명 패킷 기한 놓치면?**
→ 서명 기한이 계약서상 효력 발생 조건이면 무효. 재발행 필요. 당사자 전원 재동의.

**Q: 베트남 직원 해고 시 주의할 점?**
→ 계약 유형 확인 → 사유 없는 해고 시 보상금 의무 → 노동청 신고 필요 여부 확인.

**Q: 투자 계약에서 가장 중요한 조항은?**
→ Liquidation Preference + Anti-dilution. 이 두 조항이 Exit 시 실질 수익 결정.
