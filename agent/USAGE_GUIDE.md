## 🎯 시나리오별 사용법

### 시나리오 1: 투자자 미팅 준비

```bash
> "투자자 미팅 준비하려고 하는데 필요한 자료 정리해줘"
→ Finance Agent: 최근 재무 데이터
→ AR Agent: 수금 현황
→ Legal Agent: 주주 구조

> "Cap table 최신 버전"
→ Legal Agent: shareholder_list.md

> "최근 3개월 매출 트렌드"
→ Finance Agent: 월별 매출 집계
```

### 시나리오 2: 계약 갱신

```bash
> "다음 달 만료되는 계약 있어?"
→ Legal Agent: 계약 만료 일정

> "이 계약 조건 요약해줘"
→ Legal Agent: 주요 조항 추출

> "갱신하려면 뭐 해야 돼?"
→ Legal Agent: 갱신 프로세스 안내
```

### 시나리오 3: 긴급 캐시 이슈

```bash
> "캐시가 부족한데 언제 수금 들어와?"
→ AR Agent: 예상 입금 일정
→ Finance Agent: 현재 캐시 포지션

> "이번 주 지출 예정 얼마야?"
→ Finance Agent: Bill.com 예정 결제

> "수금 독촉 필요한 거 있어?"
→ AR Agent: 30+ days overdue 리스트
```

### 시나리오 4: 팀 온보딩

```bash
> "새 팀원이 알아야 할 프로세스 정리해줘"
→ Operations Agent: workflows.md 기반 가이드

> "우리 회사 구조 설명해줘"
→ 모든 에이전트: company.md 참조

> "주요 도구들 리스트"
→ 각 에이전트의 사용 도구 정리
```

## 🔧 고급 사용법

### 멀티 에이전트 협업 요청

```bash
> "이번 분기 전체 리포트 만들어줘 (재무, 법무, 운영 전부)"
→ Finance Agent: 분기 재무 리포트
→ Legal Agent: 분기 법무 이슈 요약
→ Operations Agent: 분기 프로젝트 진행
→ 통합 리포트 생성

> "주주총회 준비 자료 전부 만들어줘"
→ Finance Agent: 재무제표
→ Legal Agent: 주주명부, 안건
→ Operations Agent: 회의록 템플릿
```

```bash
> "지난 6개월 수금 트렌드 분석해줘"

> "국가별 비용 비교해줘"

```

### 의사결정 지원

```bash
> "베트남 오피스 확장하면 비용 얼마나 늘어?"
→ Finance Agent: 현재 베트남 비용 + 예측

> "이번 환율에 USD 송금하는 게 나을까?"
→ Finance Agent: 환율 트렌드 분석 + 추천

> "이 계약 조건이 유리해?"
→ Legal Agent: 계약 조건 분석 + 비교
```
