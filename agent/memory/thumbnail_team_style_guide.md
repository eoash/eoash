## 📋 팀의 톤 (Tone)

### 핵심 특성
- **직접 호소형**: 시청자에게 직접 말걸기 스타일
- **질문 중심**: 시청자의 호기심/불안감에 질문으로 접근 (70회 패턴)
- **공감 + 솔루션형**: 문제 인식 → 해결책 제시 (30회 패턴)
- **유머 감각**: 가볍고 친근한 태도 (ㅋㅋㅋ, 이모지)
- **협력적**: 팀원 의견을 존중하고 함께 개선

---

## 🎯 추천 카피 패턴

### 패턴 1: 직접 호소형 (추천도: ⭐⭐⭐⭐⭐)
```
당신이 [상황]이라면?
당신은 [현상] 중 하나입니다
왜 내 [도구]는 [문제]일까?
혹시 당신도 [경험]?
```

**예시**:
- "당신의 AI가 그저 착하기만 한 이유"
- "왜 내 ChatGPT는 동문서답을 할까?"
- "당신은 AI의 극히 일부분만 쓰고 있습니다"

### 패턴 2: 공감 + 솔루션형 (추천도: ⭐⭐⭐⭐)
```
[문제] 때문인가요?
해결 방법은 [숫자가지/방법]
이것만 고치면 됩니다
[솔루션]으로 해결하세요
```

**예시**:
- "AI 답변이 영 찝찝한 이유? 당신이 놓친 5가지"
- "AI가 또 이상한 소리 하네? 체크해야 할 4가지 습관"
- "AI 결과물 맨날 버리는 당신, 이것만 고치면 됩니다"

### 패턴 3: 질문형 (추천도: ⭐⭐⭐⭐)
```
[상황]인가요?
[도구]가 [현상]이 되는 이유?
[경험]하지 않으셨나요?
```

## ✨ 카피 작성 가이드

### DO (해야 할 것)
✅ 숫자 활용 (5가지, 10배, 3단계 등)
✅ 시청자에게 직접 말걸기
✅ 호기심/불안감 자극
✅ 구체적이고 명확한 언어
✅ 짧고 임팩트 있는 제목
✅ 한국식 마케팅 톤 (친근감)
✅ 문제 → 해결책 구조

### DON'T (피해야 할 것)
❌ "금쪽이" (아동혐오 표현)
❌ "가스라이팅" (부정적/법적 문제)
❌ "Stanford's" (법적 제약)
❌ 너무 긴 설명형 문장
❌ 기술 용어만 나열
❌ 뻔한 클리셰
❌ 브랜드 규칙 위반

---

## 📊 숫자 활용 패턴

팀이 선호하는 숫자 표현:
- **5가지** (가장 많이 사용)
- **10배** (생산성 강조)
- **3단계** (간단함)
- **4가지** (구체성)
- **1시간** (시간 압박)
- **90%** (통계)

## 💡 참고 예시 (실제 대화에서 수집)

### 팀이 선호하는 카피
1. "사실 당신은 AI의 극히 일부분만 쓰고 있습니다"
2. "AI가 만드는 '평균의 늪'에서 탈출하는 방법 5가지"
3. "아직도 AI 못 믿는 당신, 이 4가지만 바꿔보세요"
4. "ChatGPT 답변 100% 못 믿겠다면 지금 당장 해야 할 5가지"
5. "AI 한테 계속 실망하는 당신, 문제는 여기 있습니다"

### 팀이 거부한 카피
- "극소수만 알고 있는 AI 생산성..." (뻔함)
- "스탠포드식..." (법적 제약)
- "금쪽이 AI..." (아동혐오)
- "당신의 프롬포트가 잘못된 이유" (너무 직접적)

## 🎬 카피 작성 프로세스

1. **주제 분석**
   - 영상 제목 및 주요 콘텐츠 파악
   - 타겟 오디언스 정의

2. **초안 생성** (10-20개)
   - 위의 패턴 참고
   - 숫자 활용
   - 질문형/직접호소형 혼합

3. **평가 기준**
   - CTR 잠재력
   - 명확성 (5초 안에 이해)
   - 브랜드 톤 일관성
   - 법적/윤리적 준수

4. **팀 피드백**
   - Slack 스레드에서 투표
   - 개선 제안 반영
   - 최종 선택

## 🔄 반복 개선

팀은 자주:
- "약한 것 같아요" → 더 강한 버전 요청
- "뻔한 카피가 나올 것 같은데" → 새로운 각도 제시
- "이거 뭔가 팍 꽂히는 게 없네요" → 재작성
- "이거 어때요?" → 피드백 요청

---

## 📈 성공 지표

카피가 좋은지 판단하는 기준:
1. **CTR 잠재력**: 클릭하고 싶은가?
2. **명확성**: 5초 안에 이해되는가?
3. **차별성**: 다른 채널과 구별되는가?
4. **팀 합의**: 대부분의 팀원이 동의하는가?
5. **실행성**: 디자인으로 구현 가능한가?

---

## 🎨 디자인과의 조화

- 제목은 **임팩트** 우선
- 부제목으로 **설명** 추가
- 숫자는 **시각적으로** 강조
- 교수/전문가 이미지는 **필수** (신뢰도)
- 한국식 이모지/밈 활용 권장
